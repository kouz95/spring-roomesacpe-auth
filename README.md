## 상황 설명

- 전화 예약의 한계를 느껴 온라인 예약을 지원하기로 한다.
- 온라인 예약을 위해 로그인 기능을 제공한다.

## 요구사항

### 기능 요구사항

- [x] 로그인 기능
  - [x] 토큰을 발급
- [x] 내 정보 조회하기
  - [x] 토큰을 이용하여 본인 정보 응답하기

### API 설계

POST /login/token HTTP/1.1
accept: */*
content-type: application/json; charset=UTF-8

{

    "password": "password"
}
